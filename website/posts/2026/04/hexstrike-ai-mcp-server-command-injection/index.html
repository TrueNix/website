Check Point — HexStrike AI MCP Server Command Injection

AI relevance: This vulnerability affects MCP (Model Context Protocol) servers that power AI agent tooling ecosystems, demonstrating how command injection flaws in AI infrastructure can lead to full system compromise when agents execute malicious tool calls.

  • CVE-2025-35028 affects HexStrike AI MCP Server commit 2f3a5512
  • Command injection vulnerability in EnhancedCommandExecutor class
  • Attackers can execute arbitrary commands via API endpoints
  • Vulnerable when command-line arguments start with semicolon (;)
  • Server typically runs with root privileges in default configuration
  • Allows complete system takeover through AI agent tool execution
  • Check Point provides IPS protection for Security Gateway products

Why it matters

MCP servers form the backbone of AI agent tool ecosystems, and command injection vulnerabilities at this layer represent critical infrastructure risks. When AI agents can be tricked into making malicious tool calls, these vulnerabilities enable attackers to bypass all security controls and gain complete system access.

What to do

  • Update HexStrike AI MCP Server to patched versions
  • Implement input validation for all MCP server API endpoints
  • Run MCP servers with least privilege principles
  • Deploy network-level protections like Check Point's IPS
  • Monitor for suspicious command execution patterns

Sources:
