VulDB — MCP Data Visualization Server SQL Injection (CVE-2026-5322)

AI relevance: MCP servers are critical infrastructure for AI agent tooling, and SQL injection vulnerabilities can allow malicious actors to manipulate agent data flows and exfiltrate sensitive information.

Critical Vulnerability Details

  • CVE: CVE-2026-5322
  • Component: AlejandroArciniegas mcp-data-vis MCP Handler
  • Affected file: src/servers/database/server.js
  • Vulnerability type: SQL injection
  • CVSS severity: Critical
  • Attack vector: Remote

Why It Matters

MCP (Model Context Protocol) servers are becoming standard infrastructure for AI agent tooling. This SQL injection vulnerability allows attackers to:

  • Execute arbitrary SQL commands through the MCP handler
  • Manipulate database contents that AI agents rely on
  • Potentially exfiltrate sensitive data processed by agents
  • Compromise the integrity of data visualizations used for decision-making

What To Do

  • Immediately update to patched versions if available
  • Audit MCP server implementations for proper input validation
  • Implement parameterized queries and proper SQL injection protections
  • Monitor MCP server traffic for suspicious database operations
  • Consider using prepared statements for all database interactions
