Cisco — AI Agent Security Framework for enterprise adoption

AI relevance: Cisco's new security framework addresses critical gaps in AI agent deployment by extending zero trust principles to non-human identities, providing runtime security enforcement, and enabling safe enterprise-scale agentic workforce adoption.

At RSA Conference 2026, Cisco announced a comprehensive security framework designed to enable safe enterprise adoption of AI agents. The solution addresses three key pillars: protecting the world from agents, protecting agents from the world, and detecting/responding to AI incidents at machine speed.

Key Framework Components

  • Zero Trust for AI Agents: Extends Duo IAM capabilities to register agents and map them to human owners
  • MCP Policy Enforcement: Integrates novel MCP policy enforcement with intent-aware monitoring in Cisco Secure Access
  • AI Defense Explorer Edition: Self-service red teaming solution for AI models and applications
  • Agent Runtime SDK: Embeds policy enforcement directly into agent workflows at build time
  • LLM Security Leaderboard: Comprehensive resource for evaluating model risk and adversarial susceptibility

Enterprise Adoption Challenges

  • Visibility Gap: 85% of enterprises experiment with AI agents but only 5% deploy to production
  • Identity Management: Existing tools lack capability to enforce time-bound access for agentic identities
  • Context Understanding: Legacy security tools cannot comprehend context behind agent requests
  • Testing Limitations: Traditional scanning cannot simulate real-world agent threats with tool access

Why It Matters

AI agents represent a fundamental shift from software that answers questions to software that takes action. This transformation creates unprecedented security challenges that traditional human-centric security models cannot address. The 2025 Cisco Talos Year in Review showed attackers overwhelmingly target components that authenticate users, enforce access decisions, or broker trust between systems — a trend that will accelerate with agentic workloads.

Cisco's framework provides the first comprehensive approach to securing agentic workforces at enterprise scale, addressing both the technical security requirements and the operational governance needed for safe AI adoption.

What to Do

  • Agent Identity Management: Register AI agents in identity systems with clear human accountability
  • Fine-Grained Access Control: Assign minimal permissions for specific tasks with short durations
  • Red Team Testing: Use AI Defense Explorer to test models before production deployment
  • Runtime Enforcement: Implement policy enforcement directly in agent workflows via SDK integration
  • Continuous Monitoring: Leverage LLM Security Leaderboard for ongoing risk assessment
